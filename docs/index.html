CLI

Full control from the terminal

Manage Kerstel without leaving your workflow.

kerstel open       # launch the app
kerstel stop       # stop the app
kerstel restart    # restart
kerstel update     # pull latest & rebuild
kerstel version    # show version
kerstel uninstall  # remove everything
